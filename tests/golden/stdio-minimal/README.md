```bash

```

```bash

```

## Regenerate managed assets

After editing `mcp-builder.yaml`:

```bash
mcp-builder generate --dry-run
mcp-builder generate
```

Scaffold-once source files are preserved. Managed-file conflicts require an
explicit `--force-managed PATH`.
