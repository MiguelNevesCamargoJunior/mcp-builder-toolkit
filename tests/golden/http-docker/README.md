## Run

Streamable HTTP (binds to `127.0.0.1:8000/mcp` by default):

```bash
uv run python -m http_docker_golden.server
```

**Security note:** Prefer loopback (`127.0.0.1`). Do not expose this port
to untrusted networks without authentication and TLS at the edge.

### Container

```bash
docker build -t http-docker-golden .
docker run --rm -p 8000:8000 http-docker-golden
```
### Compose

```bash
docker compose up --build
```

explicit `--force-managed PATH`.
